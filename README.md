# sHud-Minimalist
HUD minimaliste pour FiveM

**Dependencies** : 
- es_extended
- esx_status (Optional)


**Status HUD - Health & Armour** 
- 4 bars for Health, Armour, Hunger and Thirst
- Hunger and Thirst disapear if you don't have esx_status

**Main HUD - Money and ID**
- Bank Money
- Money item or pocket money
- Your ID
